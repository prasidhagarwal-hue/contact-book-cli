# Contact Book CLI App 📒

A command-line contact management app built with Python.

## Features
- Add new contacts (name, phone, email)
- View all saved contacts
- Search contact by name
- Delete contact by name
- Data saved permanently using JSON

## Tech used
- Python 
- OOP (classes, methods)
- JSON for data persistence
- OS module for file handling

## How to run
1. Clone the repo
2. Run: python main.py

- OOP design with multiple classes across files
- JSON file handling and data persistence
- CLI app structure with while loop menu
